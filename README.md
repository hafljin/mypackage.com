# 飲食店LINEボット デモ

飲食店の営業時間を答えるLINEボットのデモアプリケーションです。

## 概要

このプロジェクトは、飲食店の問い合わせ対応を自動化するLINEボットのデモです。現在はモック（擬似機能）で実装されており、本番環境では**GPT-5 nano**を使用する予定です。

## 機能

- 営業時間の自動応答
- 定休日の案内
- 今日・明日の営業情報
- 予約に関する案内
- メニュー・アクセス情報の提供
- チャット形式のインタラクティブなUI
- クイック返信ボタン

## 技術スタック

- **React** + **TypeScript**
- **Tailwind CSS**
- **Vite**
- **Lucide React** (アイコン)
